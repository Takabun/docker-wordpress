Wordpressを用いたサイトを制作する必要があるが、デザインおよびテンプレートコーディングを他社に業務委託するために、
Dockerを用いた開発環境を提供する必要がある。以下を条件とする。

・業務委託先は開発環境と同様の状態である本番環境にSFTPでファイルをアップロードすることで納品完了とする
・利用可能なツールは docker と docker-composeのみとする
・利用可能な Docker Image は 下記の2つのいずれかのみとする
	https://hub.docker.com/_/centos/
	https://hub.docker.com/_/alpine
・SSL化やドメインについては考慮しなくても良い（URLはhttp://localhost/で良い）
